Для запуска проекта без докера:
1) Рекомендуется запуск скрипта внутри virtual ENV, дабы не засорять систему лишними библиотеками
2) Перед запуском установите все необходимые библиотеки - в терминале введите команду pip install -r requirements.txt 
(или pip3 install -r requirements.txt для linux и mac)
Команда установит библиотеки для запуска и скрипта, и телеграм-бота.
3) При первом запуске скрипта будет отсутствовать файл creds.json - это данные для подключения кода к БД. Так как данные БД могут отличаться от пользователя
к пользователю, скрипт самостоятельно предложит ввести данные для подключения к БД. В случае, если данные были введены неправильно, скрипт сообщит вам об этом,
однако файл creds.json необходимо будет удалить в ручную

Для запуска бота:
1) Запустите скрипт bot.py
2) Оповещения будут присылаться через бота @js129492_bot. После того, как вы запустите бота в самом Телеграме, нажмите кнопку "Запустить" - оповещения будут
приходить раз в 24 часа с момента запуска
